# 📊 营销活动数据分析平台 - 使用说明

## 一、概述

本平台用于分析营销活动的效果，需要上传两个数据文件进行综合分析。

---

## 二、数据文件要求

### 📋 文件1：公司购买明细

**文件格式：** Excel (.xlsx) 或 CSV (.csv)

**必需字段（表头）：**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| 活动名称 | 营销活动的名称 | 新年营销峰会 |
| 公司名称 | 购买产品的公司名称 | 腾讯科技 |
| 产品名称 | 购买的产品名称 | 企业版CRM |
| 合同订单金额/万元 | 合同总金额（单位：万元） | 50 |
| 新购合同订单金额/万元 | 新购订单金额（单位：万元） | 50 |
| 新客户合同订单金额/万元 | 新客户订单金额（单位：万元） | 0 |

**数据示例：**
```
活动名称,公司名称,产品名称,合同订单金额/万元,新购合同订单金额/万元,新客户合同订单金额/万元
新年营销峰会,腾讯科技,企业版CRM,50,50,0
新年营销峰会,阿里巴巴,高级咨询服务,80,80,80
数字化转型论坛,百度网讯,基础版CRM,30,30,30
```

**注意事项：**
- ✅ 字段名必须完全一致（包括斜杠、空格）
- ✅ 金额字段可以为 0（表示老客户续费或非新客户）
- ✅ 每一行代表一笔购买记录
- ✅ 同一公司可以有多条购买记录

---

### 📊 文件2：活动线索数据

**文件格式：** Excel (.xlsx) 或 CSV (.csv)

**必需字段（表头）：**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| 活动名称 | 营销活动的名称 | 新年营销峰会 |
| 线索数 | 活动产生的线索总数 | 580 |
| 触达企业数 | 活动触达的企业数量 | 48 |
| 合同订单转化率 | 线索转化为订单的比例 | 6.25 或 6.25% |
| 合同订单金额/万元 | 活动产生的订单总额（单位：万元） | 160 |
| 新购合同订单转化率 | 新购订单转化率 | 6.25 或 6.25% |
| 新客户合同订单金额/万元 | 新客户订单总额（单位：万元） | 110 |

**数据示例：**
```
活动名称,线索数,触达企业数,合同订单转化率,合同订单金额/万元,新购合同订单转化率,新客户合同订单金额/万元
新年营销峰会,580,48,6.25%,160,6.25%,110
数字化转型论坛,420,35,8.57%,140,8.57%,95
线上产品发布会,650,52,5.77%,150,5.77%,95
```

**注意事项：**
- ✅ 字段名必须完全一致（包括斜杠、空格）
- ✅ 转化率可以带 % 符号，也可以不带（系统会自动处理）
- ✅ 每一行代表一个活动的汇总数据
- ✅ 活动名称应与文件1中的活动名称保持一致

---

## 三、使用步骤

### 1️⃣ 准备数据文件

- 确保两个文件的表头字段名称完全符合上述要求
- 检查数据格式是否正确（数字字段不要包含文本）
- 保存为 `.xlsx` 或 `.csv` 格式

### 2️⃣ 上传文件

1. 打开分析平台页面（`index-v2.html`）
2. 在左侧框中上传 **公司购买明细** 文件
3. 在右侧框中上传 **活动线索数据** 文件
4. 等待文件读取完成（显示绿色提示和数据条数）

### 3️⃣ 开始分析

1. 点击 **"🚀 开始分析"** 按钮
2. 等待数据处理完成（数据量大时可能需要几秒钟）
3. 页面会自动滚动到分析结果区域

### 4️⃣ 查看结果

分析结果包含三个部分：

**📊 统计卡片**
- 总销售额
- 总线索数
- 触达企业数
- 平均转化率
- 购买企业数
- 活动数量
- 单个线索价值
- 客均订单金额

**🏆 Top 榜单**
- 最高消费企业
- 最佳营销活动
- 最畅销产品

**📋 详细表格**
- 企业购买排名（前20名）
- 活动效果排名（按销售额排序）
- 产品销售排名

## 四、常见问题

### Q1: 上传文件后没有显示分析结果？
**A:** 请检查：
- 表头字段名是否完全一致（包括标点符号）
- 数据是否从第一行开始（第一行必须是表头）
- 按 F12 或 Cmd+Option+I 打开控制台查看错误信息

### Q2: 字段名可以用英文吗？
**A:** 目前系统仅支持中文字段名，请按照上述标准字段名准备数据。

### Q3: 数据量有限制吗？
**A:** 理论上无限制，但数据量过大（超过10万条）可能导致浏览器处理缓慢。

### Q4: 可以只上传一个文件吗？
**A:** 可以。系统会根据已上传的文件进行分析，但只能看到部分数据（只有购买明细或只有线索数据）。

### Q5: 转化率是百分比还是小数？
**A:** 两种格式都支持。可以填写 `6.25` 或 `6.25%`，系统会自动识别。
