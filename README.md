# 营销活动数据分析平台 V2.0

## 🎯 项目简介

这是一个基于 Web 的营销活动数据分析平台，提供智能问答功能，帮助您快速洞察营销数据。

## ✨ 核心功能

### 📊 数据分析
- 企业购买行为分析
- 活动效果评估
- 产品销售统计
- 转化率分析

### 🤖 智能问答
支持自然语言查询，例如：
- "腾讯科技购买了哪些产品？"
- "新年营销峰会有哪些公司购买？"
- "企业版CRM被哪些公司购买？"
- "哪个活动效果最好？"

## 🚀 在线访问

访问地址：[在线演示](https://your-username.github.io/your-repo-name/)

## 📖 使用方法

1. 上传"公司购买明细"数据（CSV/Excel格式）
2. 上传"活动线索数据"（CSV/Excel格式）
3. 点击"开始分析"
4. 使用智能问答查询数据
